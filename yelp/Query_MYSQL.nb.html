Querying the Yelp SQL database

Load R packages.

library(tidyverse)
library(RMySQL)

The database

We have created a copy of the public Yelp Dataset and put the data into our own MySQL database. Here is how the database is structured.

Establish connection to remote database server

Load driver and create connection to MySQL server.

driver <- dbDriver("MySQL")

connection <- dbConnect(driver,
                        host ="manila.uni-paderborn.de", 
                        port =3306,
                        dbname = "yelp_db",
                        user = "student", password = "upb123!")

Query database using SQL

Execute a simple SQL query.

businesses_100 <- dbGetQuery(connection, statement = "
                  SELECT *
                  FROM business 
                  LIMIT 100")


head(as_tibble(businesses_100))
NA

Execute a more complex SQL query involving inner join and where clauses.


pizza_places_500 <- dbGetQuery(connection, statement = "
                  SELECT *
                  FROM business 
                  INNER JOIN category
                  ON business.`id` = category.`business_id`
                  WHERE category.`category` = 'pizza'
                  AND business.city = 'Phoenix'
                  LIMIT 500")

head(as_tibble(pizza_places_500))
